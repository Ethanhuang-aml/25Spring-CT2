Final prototype demonstration

This is a homemade musical instrument, which is composed of buttons, position sensor, and Flex sensor.

Through different interaction methods, a total of four notes can be played.

I also made an upgrade in sound visualization.

Documentation

1. Assemble the circuit according to the idea.

2. Cut the box so that components such as ESP32 can be embedded in it.

3. Box installed.

4. Edit the inputs in Arduino as needed.

5. Write code to process input in processing and do the mapping.

6. Hardware and software are all complete.
